TAG: H1...H6

(voltar)

Tags são utilizadas para criar títulos de diferentes níveis. O H1 é o título principal e H6é o título de nível mais baixo. Eles são usados para estruturar e hierarquizar o conteúdo da página.

TAG: P

(voltar)

A tag 'p'e'/p'

É utilizada para criar parágrafos de texto. Ela envolve um bloco de texto e define uma quebra de linha antes e depois do parágrafo.

TAG: MARK

(voltar)

A tag 'mark' é usada para destacar um trecho de texto. Geralmente, o texto envolvido pela tag 'mark' é realçado com uma cor de fundo diferente, indicando que é importante ou relevante.

TAG: SMALL

(voltar)

A tag SMALL é usada para indicar que o texto deve ser exibido em um tamanho menor do que o normal. Geralmente, é usado para notas de rodapé, legendas ou informações secundárias.

TAG: i

(voltar)

A tag i é usada para colocar o texto em itálico. Ela é usada para indicar ênfase ou destacar uma palavra ou frase.

TAG: u

(voltar)

A tag u é utilizada para sublinhar o texto. Ela pode ser usada para indicar ênfase, palavras-chave importantes ou links não visitados.

TAG: Strong

(voltar)

A tag strong é usada para destacar um trecho de texto em negrito. É usada para enfatizar a importância de um texto e pode ser interpretada como um peso maior para os motores de busca.

TAG: ul/ol/li

(voltar)

ol,ul,li, Essas tags são usadas para criar listas.

    ol: É usado para criar uma lista ordenada (numerada).
  1. Rafael
  2. Anisio
    ul: É usado para criar uma lista não ordenada (com marcadores).
  • Rafael
  • Anisio
    li: É usado para criar cada item da lista.
  1. Rafael
  2. 07
  3. Janeiro
  4. 1997

TAG: a

(voltar)

A tag a é usada para criar links em uma página:Visit W3Schools.com!

TAG: hr

hr (voltar)

é usada para criar uma linha horizontal no conteúdo da página. É comumente usada para separar seções ou divisões de conteúdo.

TAG: sub

(voltar)

sub é usada para exibir texto subscrito, ou seja, texto posicionado ligeiramente abaixo da linha de base normal. É frequentemente usado para fórmulas químicas, equações ou notas de rodapé.

TAG: sup

(voltar)

sup é usada para exibir texto sobrescrito, ou seja, texto posicionado ligeiramente acima da linha de base normal. É comumente usado para expoentes, números ordinais ou notas de rodapé.

TAG: Blockquote

(voltar)

blockquote é usada para citar um bloco de texto que é uma citação de outra fonte. O texto dentro da tag é geralmente indentificatado ou destacado para distingui-lo do restante do conteúdo.

TAG: font

(voltar)

font é usada para alterar a fonte, tamanho e cor do texto. No entanto, essa tag é considerada obsoleta no HTML5 e não é mais recomendada para uso. É preferível utilizar estilos CSS para estilizar o texto.

TAG: del

(voltar)

del é usada para indicar que um trecho de texto foi deletado ou removido. Geralmente, o texto envolvido pela tag 'del'é exibido com uma linha horizontal através dele.

TAG: abbr

(voltar)

abbr é usada para marcar uma abreviação ou sigla no texto. O atributo 'title' é usado para fornecer uma explicação ou descrição da abreviação quando o usuário passa o mouse sobre ela.
